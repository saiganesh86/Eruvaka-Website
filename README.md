Eruvaka Project

This is the main repository for the Eruvaka web application. It is built using Python and the Django framework. The project includes several different sections to handle the website functionality, such as a home page, product listings, blogs, an about us page, and a contact form.

To run this project on your local machine, you will need to have Python installed. First, set up a virtual environment and activate it. Next, make sure to install all the required dependencies. Once your setup is complete, you can start the development server using the manage.py file and view the website in your browser.
